
# Shakti
Multi thread And Single thread google dorker written in bash

![](header1.png)

## Command to run on Termux 


```
git clone https://github.com/indiancybertroops/Shakti/
```
```
cd Shakti
```
```
chmod +x *
```
```
sh setup.sh
```



## On linux :

```
git clone https://github.com/indiancybertroops/Shakti/
```

```
cd Shakti
```

```
chmod +x *
```

```
sh setup.sh
```
