#Installation :
> For linux and Termux there are different Options in Dorker kindly choose correct option
## Command to run on Termux 


```
git clone https://github.com/indiancybertroops/Shakti/
```
```
cd Shakti
```
```
chmod +x *
```
```
sh ICT-Setup.sh
```



## On linux :

```
git clone https://github.com/indiancybertroops/Shakti/
```

```
cd Shakti
```

```
chmod +x *
```

```
sh ICT-Setup.sh
```
